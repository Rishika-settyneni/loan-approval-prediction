# Loan Approval Prediction System

This is a Machine Learning based web application that predicts whether a loan will be approved or not based on user inputs.

## Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- HTML/CSS

## Project Structure
Dataset.csv
static/
templates/
app.py
loan_approval.py
loan_model.pkl
requirements.txt


## How to Run
```bash
pip install -r requirements.txt
python app.py

Description

The model is trained using historical loan data.

Flask is used to create a web interface.

The trained model is saved using pickle and loaded during prediction.
